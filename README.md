# Simulador Seguidor de Linha

Projeto voltado ao desenvolvimento de um simulador de pista para robôs seguidores de linha utilizando Python e Pygame. 

O simulador tem como objetivo permitir testes, validação e otimização de algoritmos de controle — especialmente controladores PID — antes da implementação em hardware real, reduzindo tempo de testes físicos e facilitando a análise de desempenho.

---

## Versão 1 (V1)

Nesta primeira versão, o objetivo é desenvolver um simulador contendo:

- Robô seguidor com 5 sensores frontais e 1 sensor lateral  
- 3 opções diferentes de pistas para teste  
- Modelagem do movimento baseada na dinâmica dos motores, considerando velocidade individual de cada roda  

O foco desta versão é criar uma base funcional para evolução futura do simulador, permitindo posteriormente a implementação de algoritmos mais avançados de controle e análise de dados.
